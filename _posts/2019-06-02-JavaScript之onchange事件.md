 当用户改变input输入框内容时执行一段Javascript代码：

<input type="text" onchange="myFunction()">

**定义和用法**

onchange 事件会在域的内容改变时发生。

onchange 事件也可用于单选框与复选框改变后触发的事件。

**语法**

HTML 中:

<element onchange="SomeJavaScriptCode">

JavaScript 中:

object.onchange=function(){SomeJavaScriptCode};
